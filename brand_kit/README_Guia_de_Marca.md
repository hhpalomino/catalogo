# Garage Market — Mini Brand Kit

## 1) Logos incluidos
- **Icono** (carrito + techo + GM): claro y oscuro
- **Lockup horizontal** (icono + texto): claro y oscuro

### Archivos
- PNG transparente (256/512/1024/2048)
- SVG (editable)
- PDF resumen
- Mockups con 3 fondos recomendados

## 2) Fondos recomendados (3)
- Sand: `#F5F3EF`
- Charcoal: `#2E2E2E`
- Olive: `#4F6F52`

**Regla rápida**
- Fondo claro (Sand/Blanco) → usar logo **light** (stroke charcoal, GM terracotta)
- Fondo oscuro (Charcoal/Olive) → usar logo **dark** (stroke sand, GM sand)

## 3) Clear space (espacio de seguridad)
- Mantener un margen mínimo alrededor del logo equivalente al **radio de una rueda** (aprox. 1×).
- No colocar elementos UI (botones, textos, bordes) dentro de ese margen.

## 4) Tamaños mínimos
- **Favicon**: usar el icono simplificado (este) desde 32×32.
- **Header web**: icono 28–40px de alto.
- **Documentos**: lockup 25mm o más para legibilidad.

## 5) Paleta de colores (HEX)
### Marca
- Olive (Primario): `#4F6F52`
- Terracotta (Acento): `#C26D4A`
- Charcoal (Texto/Íconos): `#2E2E2E`

### UI / Neutros
- Sand (Background): `#F5F3EF`
- White (Surface): `#FFFFFF`
- Gray 100 (Borders): `#DADADA`
- Gray 600 (Texto secundario): `#6B6B6B`

### Estados
- Success: `#2E7D32`
- Warning: `#B7791F`
- Error: `#C0392B`
- Info: `#2F6F8F`

## 6) Guía de uso UI (recomendada)
- **Botón primario**: fondo Olive `#4F6F52`, texto White `#FFFFFF`
- **Hover**: `#3F5C43`
- **Pressed**: `#334B37`
- **Links**: Olive `#4F6F52` con underline en hover
- **Precio / badge destacado**: Terracotta `#C26D4A`
- **Focus ring**: Terracotta `#C26D4A` (2px)

## 7) Tipografía sugerida
- **UI**: Inter / Manrope / System UI
- **Documentación**: Inter (títulos) + System UI (cuerpo)

## 8) Do / Don’t
✅ Usar pocas tintas, espacios amplios, fondos limpios.
❌ No aplicar degradados/sombras al logo.
❌ No rotar, estirar, ni cambiar proporciones.
